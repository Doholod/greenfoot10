PROJECT TITLE:
Greenfoot Doom Raycaster

PURPOSE OF PROJECT:
A small Doom-like first-person maze shooter made with Greenfoot 2D drawing only.
It uses a raycasting renderer, no downloaded libraries, and no external assets.

HOW TO START THIS PROJECT:
Open the project in Greenfoot and press Run.

USER INSTRUCTIONS:
W / Up      move forward
S / Down    move backward
A / D       strafe
Left/Right  turn
Q / E       turn
Space       shoot
Mouse click shoot
R           restart after win or death

Goal:
Destroy all monsters, then reach the green exit portal.
